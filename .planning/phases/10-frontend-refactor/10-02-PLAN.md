  <action>
    Replace axios API calls with Supabase queries:

    1. Change imports:
       - Remove: `import { inquiriesAPI } from '../../utils/api'`
       - Add: `import { getInquiries, updateInquiry, deleteInquiry } from '../../lib/queries'`

    2. Update fetchInquiries:
       ```javascript
       const fetchInquiries = async () => {
         setLoading(true);
         try {
           const statusFilter = filters.status === 'ALL' ? undefined : filters.status;
           const { inquiries, total, error } = await getInquiries({
             page: filters.page,
             limit: filters.limit,
             status: statusFilter,
           });

           if (error) throw error;
           setInquiries(inquiries);
           setTotalCount(total);
         } catch (error) {
           console.error('Error fetching inquiries:', error);
           toast.error('Failed to load inquiries');
         } finally {
           setLoading(false);
         }
       };
       ```

    3. Update handleStatusChange:
       - Old: `await inquiriesAPI.update(id, { status })`
       - New: `const { error } = await updateInquiry(id, { status }); if (error) throw error;`

    4. Update handleDelete:
       - Old: `await inquiriesAPI.delete(id)`
       - New: `const { error } = await deleteInquiry(id); if (error) throw error;`
  </action>
  <verify>InquiriesPage lists, updates status, and deletes inquiries</verify>
