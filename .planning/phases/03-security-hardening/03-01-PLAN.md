    Express-validator custom validator pattern:
    ```javascript
    body('newPassword')
      .notEmpty()
      .withMessage('New password is required')
      .custom((value) => {
        const result = validatePasswordStrength(value);
        if (!result.isValid) {
          throw new Error(result.errors.join('. '));
        }
        return true;
      }),
    ```
