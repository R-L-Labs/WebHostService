<task type="auto">
  <name>Task 1: Install Supabase client and create utility</name>
  <files>client/package.json, client/src/lib/supabase.js</files>
  <action>
    1. Install @supabase/supabase-js in the client workspace:
       `npm install @supabase/supabase-js --workspace=client`
    
    2. Create `client/src/lib/supabase.js` with:
       - Import createClient from @supabase/supabase-js
       - Read VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY from import.meta.env
       - Export the supabase client instance
       - Add JSDoc comments for the exported client
    
    Pattern to follow (from Supabase docs):
    ```javascript
    import { createClient } from '@supabase/supabase-js'
    
    const supabaseUrl = import.meta.env.VITE_SUPABASE_URL
    const supabaseAnonKey = import.meta.env.VITE_SUPABASE_ANON_KEY
    
    export const supabase = createClient(supabaseUrl, supabaseAnonKey)
    ```
    
    Do NOT add auth state listeners yet - that's Phase 8.
  </action>
  <verify>
    - File exists at client/src/lib/supabase.js
    - npm install succeeded (check package.json has @supabase/supabase-js)
    - No TypeScript/ESLint errors
  </verify>
  <done>Supabase client utility created and dependency installed</done>
</task>

<task type="auto">
  <name>Task 2: Configure environment variables</name>
  <files>client/.env.example, client/.env</files>
  <action>
    1. Update `client/.env.example` to add:
       ```
       VITE_SUPABASE_URL=your-project-url
       VITE_SUPABASE_ANON_KEY=your-anon-key
       ```
    
    2. Update `client/.env` with actual values from the user's Supabase project.
       - The URL format is: https://[project-ref].supabase.co
       - The anon key is the public "anon" key from Supabase dashboard (Settings → API)
    
    Note: The anon key is safe to expose client-side because RLS policies protect data.
    Do NOT use the service_role key in the frontend.
  </action>
  <verify>
    - client/.env.example has both VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY
    - client/.env exists with actual values (will be gitignored)
  </verify>
  <done>Environment variables configured for Supabase connection</done>
</task>
