    **1. Check existing users in public.users:**
    - Go to Table Editor → users
    - Note the email of an existing admin user (e.g., admin@example.com)

    **2. Create Supabase Auth user:**
    - Go to Authentication → Users
    - Click "Add user" → "Create new user"
    - Enter the SAME email as the public.users record
    - Set a password (e.g., "TestPassword123!")
    - Check "Auto Confirm User" to skip email verification
    - Click "Create user"
    - Note the user's ID (UUID) shown in the users list

    **3. Link auth user to public.users:**
    - Go to SQL Editor
    - Run this SQL (replace the UUID):
    ```sql

    ```

    **4. Verify link:**
    - Go to Table Editor → users
    - Confirm the auth_id column has the UUID for that user
