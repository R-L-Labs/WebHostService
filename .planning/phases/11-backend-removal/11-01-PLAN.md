<task type="auto">
  <name>Task 2: Create Netlify configuration</name>
  <files>netlify.toml, client/.env.example</files>
  <action>
    1. Create netlify.toml in project root:
       ```toml
       [build]
         base = "client"
         publish = "dist"
         command = "npm run build"

       # SPA redirect - all routes to index.html
       [[redirects]]
         from = "/*"
         to = "/index.html"
         status = 200
       ```

    2. Create client/.env.example documenting required variables:
       ```
       # Supabase Configuration
       # Get these from your Supabase project settings
       VITE_SUPABASE_URL=https://your-project.supabase.co
       VITE_SUPABASE_ANON_KEY=your-anon-key
       ```

    The netlify.toml handles:
    - Build from client directory
    - Publish the dist folder
    - SPA routing (React Router works on refresh)
  </action>
  <verify>
    - netlify.toml exists with correct structure
    - client/.env.example documents required variables
    - npm run build still works from root
  </verify>
  <done>Netlify configuration ready for deployment</done>
</task>
